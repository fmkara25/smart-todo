# Smart To-Do ✅

A clean, responsive **smart to-do list** built with **React + TypeScript + Tailwind + Vite**.
Includes CRUD, filtering, search, sorting, and **localStorage persistence**.

## Live Demo
https://smart-todo-8x8u2turh-furkans-projects-429979b1.vercel.app

## Features

- Add / edit / delete tasks
- Mark tasks as completed (toggle)
- Filter: All / Active / Completed
- Search by task title
- Priority: Low / Medium / High
- Due date support
- Sorting:
  - by due date
  - by priority
- Persistent storage via `localStorage`
- Clean UI + mobile-friendly layout
- Empty state message
- Demo tasks + reset demo
- Clear completed tasks

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- localStorage
